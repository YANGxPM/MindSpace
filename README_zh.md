# MindSpace —— AI 心理健康陪伴助手
**[🔗English Version](README.md)**

免费 · 即时 · 基于循证心理学的情绪支持
## 📌 项目概述
MindSpace 是一款 AI 驱动的心理健康聊天产品，旨在为学生与职场人群提供 7×24 小时、匿名、循证的情绪支持。
产品基于 Claude AI，融合 认知行为疗法（CBT） 与 积极心理学，在用户情绪压力最高的时刻，提供即时、温和、可执行的心理支持。
* 角色：产品经理 / AI 产品设计
* 周期：MVP（v1.0）
* 形态：Web 聊天机器人（后端已完成，前端开发中）

## 🎯 问题背景（Problem）
心理健康需求快速增长，但优质支持难以及时获得。
* 73% 的学生在学业高压期出现明显心理困扰
* 60% 的职场人士表示工作压力已影响心理健康
* 首次心理咨询等待 2–6 周
* 单次咨询费用 $100–300，门槛高
* 夜间、周末等高压时段几乎无可用资源

核心洞察： 大量用户并非需要长期治疗，而是在情绪失控的当下，急需 被理解、被安抚，以及简单有效的应对方法。

## 💡 解决方案（Solution）
MindSpace 提供 即时、匿名、基于心理学的 AI 情绪支持，作为传统心理咨询之前的「第一道支持层」。
核心价值
* ✅ 完全匿名：无需注册，不收集个人信息
* ✅ 7×24 小时可用：覆盖深夜与周末高压时段
* ✅ 循证方法：基于 CBT 与积极心理学
* ✅ 隐私优先：会话仅存于内存中，自动销毁  

MindSpace 明确定位为：
不是治疗替代，而是情绪缓冲与自我调节工具

## 🧠 产品设计与 AI 策略
四阶段对话框架（4-Phase Conversation Framework）
模拟真实心理支持中的「微干预」过程，避免 AI 随机输出或说教感。
1. 情绪检查（Check-in）
    * 建立安全感与信任
    * 技术：反映式倾听、情绪验证
2. 问题探索（Exploration）
    * 识别情绪来源与思维模式
    * 技术：情绪命名、共情确认
3. 心理干预（Intervention）
    * 提供可执行的 CBT 技术，例如：
        * 认知重构
        * 行为激活
        * 呼吸练习
        * 去灾难化思维
4. 收尾与强化（Closure）
    * 总结关键认知
    * 情绪复测，鼓励小行动

## ⚠️ 风险控制与伦理设计
心理健康产品的核心不是“聪明”，而是“安全”。
* 危机关键词自动检测
* 即时展示人工求助资源（988、Crisis Text Line 等）
* 明确声明 AI 身份：“我不是心理咨询师”
* 高风险场景优先引导至真人支持
原则：不夸大 AI 能力，不模糊产品边界。

## 🛠️ 技术实现（MVP）
后端
* Node.js + Express
* Claude AI（Anthropic SDK）
* 内存级会话管理
* Server-Sent Events（流式响应）

前端（规划中）
* React + Vite
* 移动端优先
* 低认知负荷、情绪友好型 UI

## 🚀 项目成果与反思
成功经验
* 结构化对话显著提升 AI 回答质量
* CBT 框架有效避免“泛安慰”与空话

* AI 必须被产品规则“约束”，而非自由发挥
* 伦理与边界本身就是产品能力的一部分
