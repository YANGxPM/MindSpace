## Common Journey Patterns

### Discovery Channels
1. **Campus/workplace posters** (40%)
2. **Word of mouth** (25%)
3. **Google search "free anxiety help"** (20%)
4. **Social media ads** (10%)
5. **HR/student services email** (5%)

### Usage Patterns
- **First-time users**: Skeptical but desperate, need quick win
- **Returning users**: Know it works, use proactively before stressful events
- **Heavy users**: 3+ sessions/week during crisis periods (finals, project deadlines)

### Success Factors
1. **Immediacy**: Help available when crisis hits
2. **Validation**: Feelings acknowledged, not dismissed
3. **Practical tools**: Techniques they can use right away
4. **Privacy**: No judgment, no record
5. **Mood improvement**: Measurable progress in single session
